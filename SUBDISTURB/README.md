
# SUBDISTURB

A Sub-domain brute forcer.
## USAGE
 ```
python3 subdisturb.py [DOMAIN] -w/--wordlist [WORDLIST]
```

For cookies.
```
python3 subdisturb.py kali.org -c/--cookies 'PHPSESSION=abcdxyz' -w/--wordlist subdomains.txt
```
